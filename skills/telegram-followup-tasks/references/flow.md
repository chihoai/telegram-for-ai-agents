# Follow-up Tasks Flow

1. Use `tasks.today` when the user asks what needs attention now.
2. Use `dialogs.list` to gather candidate chats.
3. Use `chat.read` for context when creating or suggesting a task.
4. Use `tasks.suggest` for AI suggestions.
5. Use `tasks.add` for approved or explicit tasks.
6. Use `rules.add` and `rules.run` for recurring follow-up behavior.

Prefer small, specific tasks over broad reminders.
